## Goal

Create the first on-yomi bridge vocabulary dataset for Japanese learners.

## Scope

- Add at least 50 Mandarin/Japanese on-yomi-similar vocabulary entries.
- Include Traditional Chinese, pinyin, Japanese reading/explanation, category, example sentence, and caution metadata.
- Mark false friends, pronunciation traps, and Taiwan usage notes.
- Record source/review metadata.

## Out of scope

- Bulk dictionary import.
- Etymology claims without supporting sources.
- Audio files.

## Acceptance criteria

- [ ] 50 vocabulary entries exist in the structured content format.
- [ ] Every entry has pinyin, Japanese explanation, example sentence, and caution metadata.
- [ ] Similarity is presented as a learning aid, not a guarantee.
- [ ] Any external data source has documented license status before use.
