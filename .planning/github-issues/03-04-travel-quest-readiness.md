## Goal

Build the motivation layer that keeps Japanese learners returning: Travel Quest progress, scenario readiness, and short recurring learning prompts.

## Scope

- Define and implement travel readiness by scenario: airport, transport, food, shopping, hotel, and emergency.
- Surface progress as "what you can do in Taiwan" rather than only completed lesson count.
- Add a Travel Quest structure that connects lessons, vocabulary, phrasebook, and practice.
- Add at least one short recurring format, such as Daily Taiwan Chinese, that can be rendered from structured content.
- Use local state only for v1.

## Out of scope

- Accounts or cloud sync.
- Social/community challenges.
- Certificates, discounts, or external partnerships.
- Push notifications.

## Acceptance criteria

- [ ] Learner can see readiness by travel scenario.
- [ ] Travel Quest connects at least one scenario to relevant lessons, vocabulary, phrases, and practice.
- [ ] At least one recurring short format exists, such as Daily Taiwan Chinese.
- [ ] Readiness/progress uses local browser state where state is needed.
- [ ] UI does not reduce motivation to generic streaks only.
