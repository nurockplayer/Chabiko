## Goal

Add no-login practice modes that help learners review vocabulary and travel phrases.

## Scope

- Build at least three lightweight practice modes.
- Provide immediate feedback and retry.
- Persist useful local progress if appropriate.
- Keep practice deterministic and testable.

## Out of scope

- Cloud sync.
- Spaced repetition account system.
- Speech recognition.

## Acceptance criteria

- [ ] Three practice modes are available without login.
- [ ] Practice provides immediate feedback and retry.
- [ ] Local progress persistence works where implemented.
- [ ] Tests cover scoring and retry behavior.
